Reminders Tab — Care Scheduling
3 urgency states · Progress bar logic · AI tip flow
Cards sort by urgency ratio (elapsed days ÷ watering frequency). Each card shows a color-coded progress bar, urgency badge, and two actions: "Log Watered" and "Details." A Care Suggestions section surfaces an on-demand AI tip via Claude API. The tab badge reflects total plants requiring attention.

3D Viewport
Full-width · 300pt height · drag-to-rotate (touch + mouse) · model built procedurally per plantCategory · GPU memory disposed on sheet dismiss · gradient background matches plant card color swatch.
Stat Pills Row
Horizontally scrollable · Each pill: icon 18pt, 2-line label, corner radius 13pt, gap 6pt · Visible on all tabs · Garden mode adds a Hardiness Zone pill from hardinessZone API field.
Segmented Control — Tab Visibility Rules
Potted mode: 2 tabs (Care, Facts). Garden mode: 4 tabs (Care, Facts, Garden, Companions). Garden and Companions tabs hidden via display:none when mode !== 'garden' — not removed from DOM.
Companion Dot Indicators
Compatible: #34C759 · Incompatible: #FF3B30. Companion name + reason text pulled from companionPlants[] and incompatiblePlants[] arrays in AI JSON response.

04
AI Integration & Data Architecture
JSON schemas · localStorage keys · 3D classifier map
Plant profiles are generated on-demand via Claude Sonnet. Two schema variants exist: potted mode returns 14 fields; garden mode extends with 6 additional fields covering companion planting, soil pH, bed spacing, and hardiness zone. All data persists in localStorage under namespaced keys.
Potted Mode JSON Schema — 14 Fields
{
  "commonName": string,
  "scientificName": string,
  "description": string,
  "waterFrequencyDays": number, ← drives urgency logic
  "wateringInstructions": string,
  "sunlight": string,
  "sunlightInstructions": string,
  "temperature": string,
  "soil": string,
  "pruning": string,
  "fertilizingSchedule": string,
  "difficulty": "Beginner"|"Intermediate"|"Advanced",
  "plantCategory": string, ← drives 3D classifier
  "funFacts": string[5] ← Facts tab content
}
Garden Mode — 6 Additional Fields
// Extends potted schema:
{
  "idealSoilPH": string, ← Garden tab
  "idealBedSpacing": string, ← Garden tab
  "companionPlants": [{
    "name": string,
    "reason": string
  }], ← Companions tab · green dot
  "incompatiblePlants": [{
    "name": string,
    "reason": string
  }], ← Companions tab · red dot
  "gardenConditions": string,
  "hardinessZone": string ← stat pill (garden only)
}

Dark Mode Implementation
Toggled via data-theme="dark" on <body>. All color tokens override via CSS [data-theme="dark"] attribute selector — no separate stylesheet. State persisted to verdant-theme in localStorage. Tab bar transitions from rgba(249,249,249,.94) to rgba(28,28,30,.94). Blur backdrop maintained in both modes.
